# Lems Ya Backend 

- **Echo** web framework for fast and minimal API development  
- **PostgreSQL** as database  
- **Wire** for dependency injection  
- **Docker Compose** for local development  
- **JWT-based authentication**  

# 📂 Project Structure 

```
├── cmd/
│ ├── api/ # API entrypoint
│ │ ├── main.go
│ │ ├── wire.go
│ │ ├── wire_gen.go # generated by wire
│ │ └── server/
│ │ └── echo.go
│ └── migration/ # migration runner
│
├── docker/
│ └── dev.Dockerfile
├── docker-compose.yml
├── go.mod 
├── go.sum
│
├── internal/
│ ├── configs/ # configuration loader
│ ├── domain/ # business models, requests, responses
│ ├── handlers/ # application handler
│ ├── infrastructure/ # DB and external adapters
│ ├── middlewares/ # JWT, logging, etc.
│ ├── router/ # API routes
│ ├── services/ # application logic
│ └── utils/ # helpers
```

## 🛠️ Setup & Installation

### 1. Clone the repository
```bash
git clone https://github.com/471-68-SE-Classroom/p1-final-project-backend-lems-ya.git
cd p1-final-project-backend-lems-ya
```

### 2. Copy environment file
``` bash
cp .env.example .env
```

### 3. Start services with Docker
``` bash
docker-compose up -d
```

### 4. Run migrations
```bash
make migrate-schema
```
